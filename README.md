## How It Works

1. Install agents and skills (free): `npx companies.sh add fru-dev3/companies/{company} --include company,agents,skills`
2. Get the full pack with agent instructions, vault, and prompts on [Gumroad](https://aireadyu.dev)
3. Extract to `~/Documents/AIReadyU-{Vertical}/`
4. Switch each agent from Managed to External in the Paperclip dashboard
5. Drop your real documents into the vault
