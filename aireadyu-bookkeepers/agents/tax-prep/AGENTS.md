# Tax Prep Coordinator — Setup

## Connect External Instructions

1. **Download the AI Ready U: Bookkeepers pack** from [Gumroad](https://fruverse.gumroad.com/l/aireadyu-bookkeepers)
2. **Extract to** `~/Documents/AIReadyU-Bookkeepers/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced**
5. **Switch from Managed to External**
6. **Paste this path as the root folder:**

```
~/Documents/AIReadyU-Bookkeepers/agents/tax-prep
```

7. **Hit Save**

The agent will now load its full instructions from `~/Documents/AIReadyU-Bookkeepers/agents/tax-prep/AGENTS.md`
