## 11 Bookkeeping Domains

| Domain | Folder | What Goes Here |
|--------|--------|---------------|
| Clients | `clients/` | Client roster, engagement letters, onboarding docs, satisfaction surveys, renewal pipeline |
| Bookkeeping | `bookkeeping/` | Chart of accounts, transaction logs, journal entries, month-end close checklists |
| Reconciliation | `reconciliation/` | Bank statements, reconciliation reports, discrepancy logs, sign-off records |
| Reporting | `reporting/` | P&L statements, balance sheets, cash flow reports, client financial packages |
| Payroll | `payroll/` | Payroll schedules, deduction records, quarterly filings, W-2 and 1099 files |
| Tax Prep | `tax-prep/` | Tax document collection, estimated payment schedules, CPA handoff packages |
| Billing | `billing/` | Practice invoices, time tracking logs, AR aging reports, payment records |
| Compliance | `compliance/` | Regulatory checklists, audit support packages, risk exposure logs |
| Software | `software/` | QuickBooks/Xero configs, integration docs, subscription records, client software setups |
| Marketing | `marketing/` | Referral outreach, testimonials, LinkedIn content, practice positioning docs |
| Admin | `admin/` | Scheduling notes, deadline calendar, email templates, vendor contracts |
