# Consulting Goals & OKRs

This file is the north star for the AI Ready U: Consultants system. The Managing
Director reads this on every heartbeat and ensures all 10 domain agents are working
toward these goals. Every goal maps to at least one agent.

**To customize:** Replace all example goals and OKRs with your real priorities.
The structure and agent mappings remain the same.

---

## Annual Vision

**Theme: Build a scalable, systematized consulting practice where AI manages
operations — every client tracked, every process documented, every goal measured.**

---

## 1. Revenue Growth

**Owner: pipeline, proposals, finance**

### Quarterly OKRs

**Objective: Grow consulting revenue and pipeline coverage**
- [ ] Achieve 3x pipeline coverage ratio for quarterly revenue target → `pipeline`
- [ ] Submit proposals within 48 hours of discovery calls → `proposals`
- [ ] Improve proposal win rate by tracking and analyzing win/loss patterns → `proposals`
- [ ] Maintain 30-day rolling revenue forecast → `finance`
- [ ] Identify and pursue 5 new qualified leads per month → `pipeline`
- [ ] Follow up on all stale proposals older than 14 days → `proposals`

---

## 2. Client Satisfaction

**Owner: clients, deliverables, admin**

### Quarterly OKRs

**Objective: Deliver exceptional client experiences**
- [ ] Conduct monthly check-in with every active client → `clients`
- [ ] Achieve 100% on-time deliverable completion → `deliverables`
- [ ] Send project status updates weekly for every active engagement → `deliverables`
- [ ] Maintain updated client directory with engagement history → `clients`
- [ ] Respond to all client emails within 4 business hours → `admin`
- [ ] Collect feedback at end of every engagement → `clients`

---

## 3. Delivery Excellence

**Owner: deliverables, sops, knowledge**

### Quarterly OKRs

**Objective: Systematize delivery for consistency and scale**
- [ ] Document SOPs for the 3 most common engagement types → `sops`
- [ ] Create reusable templates for status reports, kickoff decks, and final deliverables → `sops`
- [ ] Conduct post-mortem on every completed engagement → `deliverables`
- [ ] Build a library of 5+ reusable frameworks → `knowledge`
- [ ] Track time-to-delivery and identify bottlenecks → `deliverables`

---

## 4. Knowledge Building

**Owner: knowledge, content, proposals**

### Quarterly OKRs

**Objective: Build intellectual capital that accelerates the practice**
- [ ] Write 3 case studies from completed engagements → `knowledge`
- [ ] Maintain competitive intelligence file with quarterly updates → `knowledge`
- [ ] Build a research library with 10+ industry reports → `knowledge`
- [ ] Create a proposal snippet library with 20+ reusable sections → `proposals`
- [ ] Identify 3 knowledge gaps and create learning plans → `knowledge`

---

## 5. Content & Visibility

**Owner: content, pipeline, clients**

### Quarterly OKRs

**Objective: Establish thought leadership and generate inbound leads**
- [ ] Publish 2 LinkedIn articles per month → `content`
- [ ] Send monthly newsletter to clients and prospects → `content`
- [ ] Create 1 presentation deck for speaking opportunities → `content`
- [ ] Track content performance and optimize based on analytics → `content`
- [ ] Convert 2+ content-sourced leads into discovery calls → `pipeline`

---

## 6. System Bootstrap

**Owner: chief-of-staff, all agents**

### Quarterly OKRs

**Objective: Complete AI Ready U system initialization**
- [ ] Every domain agent completes initial vault review → `all agents`
- [ ] Managing Director delivers first weekly consulting dashboard → `chief-of-staff`
- [ ] All recurring routines are active and running → `chief-of-staff`
- [ ] Port processes all files in intake/ → `chief-of-staff`
- [ ] Populate the 5 emptiest vault domains with real documents → `all agents`
- [ ] User has reviewed and customized goals.md with real goals → `chief-of-staff`
