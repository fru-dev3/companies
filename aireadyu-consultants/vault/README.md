This directory contains the **vault schema** — the blueprint for the 10-domain folder structure.

The full vault — with pre-built folders, synthetic data, and ready-to-use structure — is included in the **AI Ready U: Consultants pack**.

**[Get the full vault + skills → $29](https://fruverse.gumroad.com/l/aireadyu-consultants)**

After purchase, extract to `~/Documents/AIReadyU-Consultants/`. The agents already know to look there.

```
~/Documents/AIReadyU-Consultants/
├── skills/              <- 12 tool skills (gmail, calendar, drive, etc.)
└── vault/               <- 10 consulting domains with synthetic data
    ├── clients/
    │   ├── 00_current/
    │   └── 01_prior/YYYY/
    ├── proposals/
    ├── deliverables/
    └── ...
```

## 10 Consulting Domains

| Domain | Folder | What Goes Here |
|--------|--------|---------------|
| Clients | `clients/` | Client profiles, onboarding docs, engagement history, satisfaction surveys |
| Proposals | `proposals/` | Proposals, SOWs, pricing models, win/loss records, pitch decks |
| Deliverables | `deliverables/` | Project plans, status reports, milestones, deliverable drafts, handoffs |
| Pipeline | `pipeline/` | Lead lists, deal tracking, outreach sequences, referrals, revenue forecasts |
| SOPs | `sops/` | Standard operating procedures, playbooks, templates, checklists |
| Knowledge | `knowledge/` | Frameworks, case studies, research libraries, industry insights |
| Content | `content/` | Blog posts, LinkedIn articles, newsletters, editorial calendar, decks |
| Finance | `finance/` | Invoices, expense reports, P&L statements, cash flow, tax records |
| Legal | `legal/` | MSAs, NDAs, SOWs, IP agreements, compliance docs, insurance policies |
| Admin | `admin/` | Meeting notes, travel itineraries, vendor contracts, subscriptions |
