# Listings Director — Setup

## Connect External Instructions

1. **Download the AI Ready U: E-Commerce pack** from [Gumroad](https://fruverse.gumroad.com/l/aireadyu-ecommerce)
2. **Extract to** `~/Documents/AIReadyU-Ecommerce/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced**
5. **Switch from Managed to External**
6. **Paste this path as the root folder:**

```
~/Documents/AIReadyU-Ecommerce/agents/listings-agent
```

7. **Hit Save**

The agent will now load its full instructions from `~/Documents/AIReadyU-Ecommerce/agents/listings-agent/AGENTS.md`
