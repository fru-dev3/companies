This directory contains the **vault schema** — the blueprint for the 10-domain folder structure.

The full vault — with pre-built folders, synthetic data, and ready-to-use structure — is included in the **AI Ready U: Educators pack**.

**[Get the full vault + skills → $29](https://fruverse.gumroad.com/l/aireadyu-educators)**

After purchase, extract to `~/Documents/AIReadyU-Educators/`. The agents already know to look there.

```
~/Documents/AIReadyU-Educators/
├── skills/              <- 12 tool skills (gmail, calendar, drive, etc.)
└── vault/               <- 10 educator domains with synthetic data
    ├── curriculum/
    │   ├── 00_current/
    │   └── 01_prior/YYYY/
    ├── students/
    ├── assessment/
    └── ...
```

## 10 Educator Domains

| Domain | Folder | What Goes Here |
|--------|--------|---------------|
| Curriculum | `curriculum/` | Lesson plans, unit plans, course materials, standards alignment docs, resource library |
| Students | `students/` | Student records, progress notes, family communications, IEP/504 docs, intervention logs |
| Assessment | `assessment/` | Rubrics, gradebooks, assessment records, progress reports, testing data |
| Professional Dev | `professional-dev/` | PD plans, certifications, conference records, licensure docs, growth portfolios |
| Research | `research/` | Research projects, grant applications, publications, data collection, literature reviews |
| Finance | `finance/` | Classroom budget, grant records, reimbursements, personal finance docs |
| Tech | `tech/` | Tool inventory, LMS documentation, software accounts, tech integration guides |
| Wellbeing | `wellbeing/` | Workload logs, wellness check-ins, vacation records, personal routines |
| Community | `community/` | Parent communications, event records, volunteer logs, partnership docs, newsletters |
| Admin | `admin/` | Academic calendar, compliance docs, attendance records, scheduling records |
