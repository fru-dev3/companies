## 10 Startup Domains

| Domain | Folder | What Goes Here |
|--------|--------|---------------|
| Product | `product/` | Roadmaps, feature specs, user research, PRDs, competitive analysis, release notes |
| Fundraising | `fundraising/` | Pitch decks, term sheets, cap table, investor pipeline, data room, due diligence docs |
| Hiring | `hiring/` | Job descriptions, candidate pipeline, offer letters, org chart, onboarding checklists |
| Legal | `legal/` | Incorporation docs, contracts, IP filings, equity agreements, compliance records |
| Finance | `finance/` | Financial model, P&L, burn rate reports, bank statements, investor financials |
| Marketing | `marketing/` | Positioning docs, go-to-market plans, press coverage, content calendar, growth reports |
| Customers | `customers/` | Customer list, feedback logs, NPS data, case studies, support escalations |
| Operations | `operations/` | SOPs, vendor contracts, tool subscriptions, process documentation |
| Tech | `tech/` | Architecture docs, infrastructure inventory, security policies, incident reports |
| Admin | `admin/` | Board meeting minutes, investor updates, travel records, scheduling docs |
