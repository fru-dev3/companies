# Chief of Staff — Setup

## Connect External Instructions

1. **Download the AI Ready U: Freelancers pack** from [Gumroad](https://fruverse.gumroad.com/l/aireadyu-freelancers)
2. **Extract to** `~/Documents/AIReadyU-Freelancers/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced**
5. **Switch from Managed to External**
6. **Paste this path as the root folder:**

```
~/Documents/AIReadyU-Freelancers/agents/chief-of-staff
```

7. **Hit Save**

The agent will now load its full instructions from `~/Documents/AIReadyU-Freelancers/agents/chief-of-staff/AGENTS.md`
