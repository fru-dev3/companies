1. **Download the AI Ready U: Freelancers pack** from [Gumroad](https://fruverse.gumroad.com/l/aireadyu-freelancers)
2. **Extract to** `~/Documents/AIReadyU-Freelancers/`

4. **Click Advanced**
5. **Switch from Managed to External**
6. **Paste this path as the root folder:**

```

```

7. **Hit Save**
