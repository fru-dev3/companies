## 9 Business Domains

| Domain | Folder | What Goes Here |
|--------|--------|---------------|
| Clients | `clients/` | Client records, communication logs, contracts, satisfaction surveys, renewal notes |
| Projects | `projects/` | Project briefs, timelines, deliverables, status reports, change orders, post-mortems |
| Finance | `finance/` | Invoices, payment records, expense receipts, tax filings, profit reports, rate cards |
| Marketing | `marketing/` | Portfolio pieces, case studies, proposal templates, profile copy, testimonials |
| Skills | `skills/` | Certifications, learning plans, course notes, skill assessments, market rate research |
| Legal | `legal/` | Contract templates, signed agreements, NDAs, IP assignments, scope-of-work docs |
| Wellbeing | `wellbeing/` | Time logs, workload reports, vacation records, routine tracking, burnout check-ins |
| Pipeline | `pipeline/` | Prospect list, proposal drafts, follow-up logs, win/loss analysis, referral records |
| Admin | `admin/` | Tool subscriptions, scheduling docs, email templates, administrative records |
