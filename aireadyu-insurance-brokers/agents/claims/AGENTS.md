# Claims Support Director — Setup

## Connect External Instructions

1. **Download the AI Ready U: Insurance Brokers pack** from [Gumroad](https://fruverse.gumroad.com/l/aireadyu-insurance-brokers)
2. **Extract to** `~/Documents/AIReadyU-Insurance-Brokers/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced**
5. **Switch from Managed to External**
6. **Paste this path as the root folder:**

```
~/Documents/AIReadyU-Insurance-Brokers/agents/claims
```

7. **Hit Save**

The agent will now load its full instructions from `~/Documents/AIReadyU-Insurance-Brokers/agents/claims/AGENTS.md`
