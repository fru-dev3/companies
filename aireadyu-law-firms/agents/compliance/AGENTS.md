# Ethics & Compliance Director — Setup

## Connect External Instructions

1. **Download the AI Ready U: Law Firms pack** from [Gumroad](https://fruverse.gumroad.com/l/aireadyu-law-firms)
2. **Extract to** `~/Documents/AIReadyU-Law-Firms/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced**
5. **Switch from Managed to External**
6. **Paste this path as the root folder:**

```
~/Documents/AIReadyU-Law-Firms/agents/compliance
```

7. **Hit Save**

The agent will now load its full instructions from `~/Documents/AIReadyU-Law-Firms/agents/compliance/AGENTS.md`
