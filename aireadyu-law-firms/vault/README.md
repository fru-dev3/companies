## 10 Legal Domains

| Domain | Folder | What Goes Here |
|--------|--------|---------------|
| Cases | `cases/` | Active matter files, case timelines, status reports, task assignments, client summaries |
| Documents | `documents/` | Contracts, briefs, motions, demand letters, agreements, correspondence drafts |
| Research | `research/` | Case law memos, statutory analysis, regulatory reviews, research libraries |
| Intake | `intake/` | Intake forms, conflict check records, engagement letters, new matter setup docs |
| Docket | `docket/` | Court deadlines, filing calendars, hearing schedules, statute of limitations trackers |
| Billing | `billing/` | Time entries, LEDES invoices, trust account records, AR aging, realization reports |
| Clients | `clients/` | Client profiles, communication logs, satisfaction records, retention notes |
| Compliance | `compliance/` | Bar compliance records, CLE credit logs, conflict records, ethics documentation |
| Bizdev | `bizdev/` | Referral network contacts, speaking opportunities, thought leadership content, prospect pipeline |
| Admin | `admin/` | Scheduling notes, court appearance logistics, travel records, vendor contracts |
