# Chief Learning Officer — Setup

## Connect External Instructions

1. **Download the AI Ready U: Life pack** from [Gumroad](https://fruverse.gumroad.com/l/aireadyu-life)
2. **Extract to** `~/Documents/AIReadyU-Life/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced**
5. **Switch from Managed to External**
6. **Paste this path as the root folder:**

```
~/Documents/AIReadyU-Life/agents/learning
```

7. **Hit Save**

The agent will now load its full instructions from `~/Documents/AIReadyU-Life/agents/learning/AGENTS.md`
