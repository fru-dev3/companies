# Leasing Director — Setup

1. **Download the AI Ready U: Property Managers pack** from [Gumroad](https://fruverse.gumroad.com/l/aireadyu-property-managers)
2. **Extract to** `~/Documents/AIReadyU-Property-Managers/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced**
5. **Switch from Managed to External**
6. **Paste this path as the root folder:**

```
~/Documents/AIReadyU-Property-Managers/agents/leasing-agent
```

7. **Hit Save**
