## 10 Property Domains

| Domain | Folder | What Goes Here |
|--------|--------|---------------|
| Properties | `properties/` | Property profiles, condition reports, improvement logs, valuation records, portfolio summaries |
| Tenants | `tenants/` | Tenant files, lease agreements, move-in/out reports, payment history, communication logs |
| Maintenance | `maintenance/` | Work orders, repair records, preventive maintenance schedules, contractor reports |
| Leasing | `leasing/` | Vacancy listings, applications, screening reports, move-in checklists, market comps |
| Finance | `finance/` | Rent rolls, P&L by property, expense records, owner statements, tax filings |
| Legal | `legal/` | Lease templates, eviction records, legal correspondence, fair housing docs, compliance records |
| Vendors | `vendors/` | Contractor list, service contracts, bid records, performance reviews, payment records |
| Insurance | `insurance/` | Policy documents, claims records, coverage summaries, renewal schedules |
| Compliance | `compliance/` | Inspection reports, permits, safety certificates, code violation records, compliance checklists |
| Admin | `admin/` | Owner communications, scheduling docs, meeting notes, administrative records |
