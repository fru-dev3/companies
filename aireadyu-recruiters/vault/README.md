This directory contains the **vault schema** — the blueprint for the 10-domain folder structure.

The full vault — with pre-built folders, synthetic data, and ready-to-use structure — is included in the **AI Ready U: Recruiters pack**.

**[Get the full vault + skills → $59](https://fruverse.gumroad.com/l/aireadyu-recruiters)**

After purchase, extract to `~/Documents/AIReadyU-Recruiters/`. The agents already know to look there.

```
~/Documents/AIReadyU-Recruiters/
├── skills/              <- 12 tool skills (gmail, calendar, drive, etc.)
└── vault/               <- 10 recruiting domains with synthetic data
    ├── sourcing/
    │   ├── 00_current/
    │   └── 01_prior/YYYY/
    ├── outreach/
    ├── screening/
    └── ...
```

## 10 Recruiting Domains

| Domain | Folder | What Goes Here |
|--------|--------|---------------|
| Sourcing | `sourcing/` | Talent pool lists, sourcing notes, LinkedIn searches, passive candidate research |
| Outreach | `outreach/` | Outreach message templates, multi-touch sequences, response rate tracking, A/B test results |
| Screening | `screening/` | Resume review notes, skills assessment templates, phone screen scorecards, shortlists |
| Pipeline | `pipeline/` | ATS stage tracking, funnel analytics, bottleneck reports, recruiting dashboards |
| Clients | `clients/` | Client records, requisition briefs, status update logs, satisfaction notes |
| Job Descriptions | `job-descriptions/` | JD templates, role specs, salary benchmarks, competency frameworks |
| Offers | `offers/` | Offer letters, compensation benchmarks, negotiation notes, decline analysis records |
| Onboarding | `onboarding/` | Post-placement onboarding logs, 30/60/90-day check-in notes, guarantee period trackers |
| Bizdev | `bizdev/` | Prospect outreach records, client pitch decks, case studies, referral network contacts |
| Admin | `admin/` | Scheduling templates, interview coordination notes, email triage logs, administrative records |
