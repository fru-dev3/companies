# Admin Director — Setup

## Connect External Instructions

1. **Download the AI Ready U: Small Business pack** from [Gumroad](https://fruverse.gumroad.com/l/aireadyu-smb)
2. **Extract to** `~/Documents/AIReadyU-SMB/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced**
5. **Switch from Managed to External**
6. **Paste this path as the root folder:**

```
~/Documents/AIReadyU-SMB/agents/admin-agent
```

7. **Hit Save**

The agent will now load its full instructions from `~/Documents/AIReadyU-SMB/agents/admin-agent/AGENTS.md`
