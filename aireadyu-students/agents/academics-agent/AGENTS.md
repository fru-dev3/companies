# Academics Director — Setup

## Connect External Instructions

1. **Download the AI Ready U: Students pack** from [Gumroad](https://fruverse.gumroad.com/l/aireadyu-students)
2. **Extract to** `~/Documents/AIReadyU-Students/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced**
5. **Switch from Managed to External**
6. **Paste this path as the root folder:**

```
~/Documents/AIReadyU-Students/agents/academics-agent
```

7. **Hit Save**

The agent will now load its full instructions from `~/Documents/AIReadyU-Students/agents/academics-agent/AGENTS.md`
