## 9 Student Domains

| Domain | Folder | What Goes Here |
|--------|--------|---------------|
| Academics | `academics/` | Transcripts, syllabi, assignment records, grade reports, study plans, course notes |
| Research | `research/` | Research papers, citations, literature reviews, thesis drafts, advisor feedback |
| Career | `career/` | Resume, cover letters, application tracker, networking contacts, interview notes |
| Finance | `finance/` | Financial aid letters, loan documents, budget tracker, scholarship records, tuition invoices |
| Health | `health/` | Health records, insurance cards, appointment history, wellness logs, campus resources |
| Social | `social/` | Club memberships, event calendar, volunteer records, leadership roles, activity log |
| Skills | `skills/` | Certifications, course completions, learning plans, skill assessments, portfolio pieces |
| Housing | `housing/` | Lease agreements, utility records, housing applications, roommate agreements, move-in docs |
| Admin | `admin/` | Academic calendar, registration records, application deadlines, scheduling docs |
